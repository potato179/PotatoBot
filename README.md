## 감자봇의 상태를 봐주세요
감자봇의 상태에 따라 사용 가능한 기능이 제한될 수 있습니다.
* 온라인(Online): 정상운영중; 모든 기능 이용 가능
* 자리비움(Idle): 정상운영중; 일부 기능 불안정
* 다른용무중(DND): 테스트 또는 디버깅중
* 오프라인: 오류 또는 미실행중

## 기본 사용법
감자봇을 사용하기 위해서는 사용자의 디스코드 서버에 감자봇이 초대되어있어야 하며 초대는 위 링크에서 하실 수 있습니다.   감자봇의 모든 명령어는 `감자야`로 시작합니다. 감자봇과 얘기하고 싶으실 경우에는 감자봇이 메시지 보내기 권한을 가진 채널에서 `감자야 <할 말>`로 말을 걸어보세요!   감자봇과 대화하기 위해서는 기본적으로 감자봇에게는 `채팅 채널 읽기 및 음성 채널 보기, 메시지 보내기, 링크 첨부, 파일 첨부, 외부 이모티콘 사용하기, 반응 추가하기, 메시지 삭제하기, 음성채널 연결 및 말하기` 권한이 필수적으로 있어야 합니다.

### 예시 명령어
```감자야 도움
감자야 안녕
감자야 주사위
감자야 개그해줘
```   
#### 참고
감자봇은 구글 Dialogflow에서 학습을 합니다. 현재는 테스트 중이지만 기능화되면 다양한 명령어로 채팅할 수 있습니다. 또 아래 명령어를 통해 많은 편의기능을 이용하실 수 있습니다.
